Übungsaufgaben

Berechne die Leerlaufspannung UL eines Konzentrationselements aus Cu / Cu 2+ mit den Konzentrationen
c1 (Cu2+)=0, 6 mol / l
c2 (Cu2+)=0, 03 mol / l

Lösung einblenden
UL = 0, 059/2 ⋅ lg (c1 / c2) = 0, 04 V Lösung ausblenden

Berechne das Redoxpotential einer Fe2+ / Fe3+-Halbzelle mit c(Fe2+) = 0, 002 mol / l und c(Fe3+)=1, 000 mol / l

Lösung einblenden
E = 0, 77 V + 0, 059 / 1 lg (1, 000 / 0, 002) = 0, 77 V + 0, 16 V = 0, 93 V Lösung ausblenden

Berechne die Potentialdifferenz für ein Galvanisches Element aus einer Magnesiumhalbzelle c(MgSO4)=0, 03 mol/l und einer Goldhalbzelle mit c(AuCl3)=0, 8 mol/l
Hinweis: Konzentration von Feststoffen ist immer 1

Lösung einblenden
EAu = 0, 056 / 3 ⋅ lg(0, 5 / 1) = 1, 42 V
EMn = 0, 059 / 2 ⋅ lg (0, 03 / 1) - 2, 38 V = -2, 42 V
(EAu) + (EMn) = 3, 80 VLösung ausblenden

Berechne die Konzentration der Silbernitratlösung einer Silberhalbzelle, die ein Redoxpotential von 0, 52 V aufweist.

Lösung einblenden
0, 52 V = 0, 052 / 1 ⋅ lg (c(Au+)/c(Au)) + 0, 80 V
c(Au+) = 1, 8 ⋅ 10-5Lösung ausblenden
